# Quick WoW Talents Addon

Personal/manual World of Warcraft addon scaffold for Quick WoW Talents.

## Goal

Keep this intentionally basic:

- no Warcraft Logs calls from inside WoW
- no distribution pipeline yet
- bundled data only
- manual install/share while the idea is validated

## Local install

Copy the `QuickWoWTalents/` folder into:

```text
World of Warcraft/_retail_/Interface/AddOns/QuickWoWTalents
```

Then reload WoW and run:

```text
/qwt
```

## Current status

Scaffold only. The addon loads, registers `/qwt`, and prints a placeholder message.
